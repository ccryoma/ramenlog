# 概要
ラーメン店の検索、およびレビューを共有できるサービスです。

## URL

## 機能一覧
* 会員登録機能
* パスワード再設定機能
* リメンバーミー
* ログイン情報のセッション保持
* プロフィール編集機能(テストユーザーは利用不可)
* 会員一覧表示機能
* メール配信機能
* 店舗登録機能
* 店舗に対するレビュー投稿機能
* 画像投稿機能
* 店舗検索機能(一覧表示/地図表示)
* タグ検索機能
* テストログイン機能(お試し利用)
* ページネーション
* パンくずリスト
